* LiterAlura 

Projeto em Java que busca livros na API pública [Gutendex](https://gutendex.com) e salva os dados em um banco de dados PostgreSQL.  
Com ele, é possível montar um catálogo de livros com informações como título, autor, idioma e número de downloads.

* Tecnologias usadas

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Jackson
- Gutendex API

* Funcionalidades

- Buscar livros por título (usando a API)
- Salvar livros e autores no banco de dados
- Evitar salvar dados duplicados
- Listar todos os livros cadastrados
- Listar todos os autores
- Ver autores vivos em um ano específico
- Listar livros por idioma

* Como rodar o projeto

1. Clone o repositório: git clone https://github.com/julirodrigues10/LiterAlura.git

OBS: 
 Configure o banco de dados PostgreSQL no application.properties.

 Rode a aplicação no IntelliJ ou pelo terminal.
